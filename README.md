# OAuth-POC

Steps to test:
1. Run the spring boot application
2. Hit http://localhost:8080/login in browser
3. Follow the steps to login with google
4.  Hit http://localhost:8080/user --> Gives the details of logged in user
5.  Hit http://localhost:8080/token --> Gives the JWT
6.  Hit http://localhost:8080/logout --> Logs the user out
